Install
------------------------------------------------------------------------------

``shsk_dot_claude`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install shsk-dot-claude

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade shsk-dot-claude
